## 🎓 How the Fix Works

### The 3-Layer System

```
┌─────────────────────────────────────────────────┐
│  USER QUERY                                     │
│  "What is the primary endpoint metric?"         │
└────────────────────┬────────────────────────────┘
                     ↓
       ┌─────────────────────────────┐
       │ LAYER 1: Vector Search      │
       │ (Semantic similarity)       │
       │ - Fast, accurate when close │
       └──────────┬──────────────────┘
                  ↓
        Results > 3 chunks? ──YES──┐
                  │                │
                  NO               ↓
                  ↓         ┌─────────────────┐
       ┌──────────────────┐ │ LAYER 3:        │
       │ LAYER 2: Keyword │ │ Context Expand  │
       │ Fallback Search  │ │ (NEXT rels)     │
       │ - Reliable       │ └────┬────────────┘
       │ - Keyword-based  │      ↓
       └──────┬───────────┘ Found results?
              ↓             ↓
         Found results? ─── YES ──┐
              │                   │
              NO                  ↓
              ↓          ┌─────────────────┐
         ┌──────────┐    │ FORMAT ANSWER   │
         │ Return   │    │ + SOURCE CITE   │
         │ Empty    │    │ + SEND TO USER  │
         │ (LLM     │    └────────┬────────┘
         │ fallback)│             ↓
         └──────────┘      ┌──────────────┐
              ↓            │ ✅ ANSWER    │
              ↓            │ DELIVERED    │
              ↓            └──────────────┘
              └────────────────────┘
                                   ↓
                           RETRY same
                           layer down
```
